## [1.0.0] - 2026-05-14

### Added / 新增

- Single-file AI API documentation portal (`index.html`) / 单文件 AI API 文档门户
- Three-column layout: collapsible sidebar, main content, code panel / 三栏布局：可折叠侧栏、文档内容、代码面板
- Five documentation sections: Introduction, Authentication, Chat Completions, Models, Error Handling / 五大文档章节
- Dark/light theme toggle with `localStorage` persistence / 深色/浅色主题切换，选择持久化
- Stripe-inspired design system (purple accent `#635BFF`, blue-tinted shadows, conservative radii) / 参照 Stripe 设计风格
- Multi-language code tabs: Python, JavaScript, cURL / 多语言代码示例 Tab
- Regex-based syntax highlighting (JSON, Python, JavaScript, Bash) / 基于正则的语法高亮
- One-click copy button with visual feedback / 一键复制按钮
- Smooth scroll navigation with active link tracking / 平滑滚动导航与活跃链接追踪
- Responsive design: code panel collapses at ≤1024px, sidebar drawer at ≤768px / 响应式布局
- API endpoint blocks with HTTP method badges, parameter tables, response documentation / API 端点块
- Info boxes: info, success, warn, error variants / 信息提示框
- Netlify deployment configuration (`netlify.toml`) with security headers / Netlify 部署配置与安全头
- GitHub Pages auto-deploy workflow (`.github/workflows/deploy.yml`) / GitHub Pages 自动部署工作流
- Project documentation: README, CONTRIBUTING, CHANGELOG, LICENSE, .gitignore / 项目文档集
